# 📰 Retro Pixel News Portal

A lightweight, high-frequency live news portal featuring an interactive 8-bit mascot system, responsive theme styling, and strict anti-paywall filtering. Built entirely in vanilla HTML, CSS, and modern asynchronous JavaScript.

## 🚀 Key Features

*   **Live Multi-Stream Feed:** Synchronizes dual real-time public query channels to pull breaking updates across Tech, Software Engineering, FinTech, AI, Sports Analytics, and Geopolitical Defense.
*   **Zero-Friction Link Delivery:** Automatically bypasses `X-Frame-Options` and firewall connection barriers by routing pure article destination paths into pristine browser instances.
*   **Automatic Paywall & Discussion Scrubbing:** Uses precise backend validation regex to instantly discard premium registration walls (e.g., *NYTimes, Bloomberg, WSJ, Financial Times, TechReview, Washington Post*) and filters out standard community text discussions (`Ask HN`, `Show HN`).
*   **Reactive 8-Bit Mascot Loop:** Features an autonomous wandering pixel-duck that communicates state vectors with a side-peeking anime-character. Includes proximity collision detection that triggers custom grid array face configurations upon overlap.
*   **CRT Look & Feel:** Includes custom-rendered block cursors, heavy box-shadow borders, layout marquee headers, and scanline effects without external asset dependencies.

## 🛠️ Tech Stack

*   **Frontend Architecture:** Structural HTML5, Semantic Elements.
*   **Styling Engine:** Raw CSS3, Native Variable Trees (`:root`), Custom Embedded SVG cursors, Pixelated Image-Rendering Interpolation, Flexible Grid Matrix Layers.
*   **Runtime Logic:** Vanilla ECMAScript 6+, Promises, Coordinate Collision Handlers, State Change Triggers.
*   **Data Sourcing:** RESTful Search APIs.
